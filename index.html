Algorithm Stuff: My Atoi

String to Integer (atoi).

读入一个字符串，将它转换为 int 型返回。

处理过程：先忽略前 n 个空格，然后读入可选的符号（＋或－），最后读入数字，直到字符串末尾或出现非数字字符为止。
如果结果溢出，上溢返回INT_MAX；下溢返回INT_MIN。

后记： C 自带的 isspace 和 isdigit 效率好像比手动判断要高不少……不过加了优化参数后就没区别了吧
